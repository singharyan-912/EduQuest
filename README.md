# EduQuest– Learn. Play. Master.

EduQuest is a gamified web-based learning platform designed for students of Class 10 and 12.  
It transforms traditional NCERT learning into an interactive experience using quizzes, games, and real-time analytics.

---

##  Features

###  Gamified Learning
- Earn XP and level up by completing chapters
- Interactive quizzes and challenges
- Engaging learning experience

###  Full NCERT Coverage
- Structured syllabus for Class 10 & 12
- Subjects: Physics, Chemistry, Biology, Mathematics
- Chapter-wise learning modules

###  Interactive Learning
- Concept-based learning with examples
- Visual understanding and step-by-step explanations

###  Advanced Quiz System
- 20+ MCQs per chapter
- Different difficulty levels
- Instant feedback and explanations

###  Mini Games (Phaser.js)
- Physics-based games (projectile motion)
- Learning through gameplay
- Score and reward system

###  Analytics Dashboard
- Track progress and performance
- Subject-wise insights
- XP and level system

###  Parent Dashboard
- Monitor student progress
- Identify strengths and weaknesses
- View analytics and activity

---

##  Tech Stack

- **Frontend:** React (Vite) + TypeScript  
- **Styling:** Tailwind CSS  
- **Backend & Database:** Supabase  
- **Game Engine:** Phaser.js
